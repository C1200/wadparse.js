# TMD Content

## `TMDContent.id`

-   Type: **number**

Content ID.

## `TMDContent.idx`

-   Type: **number**

Index.

## `TMDContent.type`

-   Type: **string**

Type.

One of:

-   `"normal"`
-   `"dlc"`
-   `"shared"`

## `TMDContent.size`

-   Type: **bigint**

Size.

## `TMDContent.hash`

-   Type: **string**

SHA1 Hash.
