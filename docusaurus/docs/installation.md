# Installation

## Prerequisites

-   [NodeJS](https://nodejs.org/) version 15 or later

## Installation

<Tabs groupId="packageman">

        </CodeBlock>
    </TabItem>
    <TabItem value="yarn" label="Yarn">

    <TabItem value="pnpm" label="pnpm">
